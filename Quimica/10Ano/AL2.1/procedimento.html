Procedimento

  1. Escolher dois líquidos diferentes, de entre os disponíveis.
  2. Usando um conta-gotas diferente para cada um, adicionar a um tubo de ensaio o mesmo número de gotas de cada líquido. Se ambos os líquidos forem incolores, adicionar corante ao 1º líquido vertido.
  3. Agitar vigorosamente o tubo de ensaio.
  4. Registar se estão visíveis duas camadas ou apenas uma.
